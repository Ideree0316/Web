Catch the Box – Code Explanation

1. HTML Structure

HTML defines the structure of the webpage. Inside the body, there is a navigation button, a container which holds explanatory text, and a preview image for the game.

2. Container

The #container element groups the main content and uses CSS Flexbox for a two-column layout. Background, border-radius, and box-shadow create a card-like appearance.

3. Text Section

Headings, paragraphs, and lists are used to explain the game mechanics and structure in a readable, accessible way.

4. Image

The image is a visual preview of the game. CSS controls its size and presentation.

5. CSS Styling

  • Set fonts and background color
  • Center the container
  • Use Flexbox for layout
  • Apply shadows and rounded corners

6. JavaScript Navigation

JavaScript handles page navigation. The button above uses a direct location change; external script assignment.js handles navigation back to the assignment.
